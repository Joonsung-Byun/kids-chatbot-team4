# Frontend

아이들을 위한 챗봇 애플리케이션의 프론트엔드입니다.

## 기술 스택

- React.js / Next.js (추후 선택)
- TypeScript
- CSS/SCSS 또는 styled-components
- Axios (API 통신)

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm start

# 빌드
npm run build
```

## 환경 변수

`.env` 파일을 생성하고 다음과 같이 설정하세요:

```
REACT_APP_API_URL=http://localhost:3001
REACT_APP_ENVIRONMENT=development
```
